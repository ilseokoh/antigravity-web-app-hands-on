# 사전준비

Antigravity의 실행 환경은 결국 사용자의 로컬환경(PC, Mac)입니다. 로컬에서 Antigravity가 움직이려면 도구가 필요합니다. 코드를 실행, Google Cloud 작업, 커멘드 라인 작업 등 **실습에 필요한 아래 도구들을 실습전에 설치해 오시면 시간을 아낄 수 있습니다**. 설치가 되었는지 확인해보고 설치되어있다면 넘어가도 됩니다.
